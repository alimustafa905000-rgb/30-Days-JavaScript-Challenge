# 30 Days JavaScript Challenge 🚀

This repository contains my **30 Days JavaScript Challenge** where I build small projects daily to improve my JavaScript, HTML, and CSS skills.  
The goal of this challenge is to strengthen my problem-solving abilities and enhance my frontend development skills through practical projects.

## About
In this challenge, I am building one project each day using:
- **HTML5**
- **CSS3**
- **JavaScript (ES6+)**

Each project focuses on different concepts such as DOM manipulation, events, logic building, API handling, and UI design.

## Projects Completed

### Day 1 - Tic Tac Toe 🎮
A simple Tic Tac Toe game with player turns and winner detection.

### Day 2 - Rock Paper Scissors ✊✋✌️
A fun game where users play against the computer.

### Day 3 - To-Do App 📝
A task management app to add and delete daily tasks.

### Day 4 - Quiz App ❓
An interactive quiz application with score tracking.

### Day 5 - Random Password Generator 🔐
Generates strong random passwords.

### Day 6 - Stopwatch ⏱️
A stopwatch with start, pause, and reset functionality.

### Day 7 - Calculator 🧮
A functional calculator for basic arithmetic operations.

### Day 8 - BMI Calculator ⚖️
Calculates Body Mass Index based on user input.

### Day 9 - Weather App 🌦️
Fetches real-time weather data using the wttr.in API.
