 # BMI Calculator 

A simple and responsive BMI Calculator built using HTML, CSS, and JavaScript. This project helps users calculate their Body Mass Index (BMI) by entering height and weight.

# Features
Calculate BMI instantly
User-friendly interface
Shows BMI result
Displays health status:
Underweight
Normal Weight
Overweight
Obese
Responsive design
Simple and clean UI

# Technologies Used
HTML5
CSS3
JavaScript (ES6)
 
# How It Works

The user enters:

Height in centimeters (cm)
Weight in kilograms (kg)

The app calculates BMI using the formula:

BMI = Weight (kg) / Height² (m²)

Then it displays the BMI value and health category.
