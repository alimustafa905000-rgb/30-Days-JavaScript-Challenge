# Age Calculator 🎂

A simple and interactive **Age Calculator** built using **HTML**, **CSS**, and **JavaScript**.  
This app allows users to select their date of birth and instantly calculate their current age.

## Features
- Select date of birth using date input
- Calculate exact age in years
- Input validation for empty fields
- Simple and user-friendly interface
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)

 

## How It Works
1. User selects their birth date.
2. JavaScript gets the selected date.
3. The app compares it with the current date.
4. It calculates the age.
5. Displays the result on the screen.
