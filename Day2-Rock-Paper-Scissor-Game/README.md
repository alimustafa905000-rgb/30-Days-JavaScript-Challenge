✊✋✌️ Rock Paper Scissors Game

A simple Rock Paper Scissors game built using HTML, CSS, and JavaScript.
This project is part of my 30 Days JavaScript Challenge (Day 2).

🚀 Features
User vs Computer gameplay
Random computer choice generation
Live score tracking for both user and computer
Dynamic result messages (Win / Lose / Draw)
Simple and interactive UI using DOM manipulation


🎮 How to Play
Click on Rock, Paper, or Scissors
Computer will randomly choose its option
Game will decide the winner based on rules:
Rock beats Scissors
Paper beats Rock
Scissors beats Paper
Scores update automatically after each round

🧠 Game Logic
If both choices are the same → Draw
Otherwise:
Rock vs Paper → Paper wins
Paper vs Scissors → Scissors wins
Scissors vs Rock → Rock wins
