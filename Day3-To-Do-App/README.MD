
A simple and interactive To-Do List Application built using HTML, CSS, and JavaScript.
This project is part of my 30 Days JavaScript Challenge (Day 3).

Features:

Add new tasks
Mark tasks as completed
Delete tasks
Save tasks in browser using Local Storage
Tasks remain saved even after refreshing the page
Clean and responsive UI
Smooth animations and hover effects


How It Works:

Enter your task in the input field
Click the Add button to create a task
Click on a task to mark it as completed
Click the Delete button to remove a task
All tasks are automatically stored in Local Storage
