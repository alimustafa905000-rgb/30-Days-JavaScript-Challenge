# Quiz App 

A simple and interactive Quiz Application built using (HTML, CSS, and JavaScript).  
This project is part of my 30 Days JavaScript Challenge (Day 4).

# Features:

- Multiple choice questions
- Next button to move to the next question
- Score tracking system
- Highlights correct answers in green
- Highlights wrong answers in red
- Shows final score at the end
- Restart quiz option

# Technologies Used 

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

# How It Works:

1. The quiz starts automatically.
2. One question is shown at a time.
3. User selects an answer.
4. Correct answer is highlighted.
5. Score updates if the answer is correct.
6. User clicks **Next** for the next question.
7. At the end, the total score is displayed.
8. User can restart the quiz.
